# CSS Project – Web Development Exercise

This repository contains the full solution for the Web Development assignment.
The project includes four different implementations of the same responsive layout, each using a different styling method.

------------------------------------------------------------
Project Structure
------------------------------------------------------------

1. manual.html
   A fully manual implementation using CSS Plain without frameworks or AI assistance.

2. plain.html
   A version generated using GitHub Copilot, also using CSS Plain.

3. flex.html
   An implementation using CSS Flexbox, generated with Copilot while keeping the same class names.

4. bootstrap.html
   An implementation using Bootstrap, generated with Copilot, using Bootstrap grid utilities.
